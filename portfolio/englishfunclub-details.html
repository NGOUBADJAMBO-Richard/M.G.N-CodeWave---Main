Catégorie

E-learning

À Propos du Projet

English Fun Club est une plateforme d'e-learning innovante dédiée à l'enseignement de l'anglais. Nous avons développé un système complet avec cours interactifs, quiz, vidéos et suivi de progression.

Défis & Solutions

Défi : Créer une Plateforme Pédagogique Complète

Besoin d'un système d'apprentissage robuste avec gestion des utilisateurs, des cours et des évaluations.

Solution : Full-Stack E-learning Scalable

React frontend interactif, Node.js backend performant, MongoDB pour les données, système de paiements intégré.

Fonctionnalités Principales

  • Cours Structurés Interactifs
  • Quiz & Évaluations Automatisés
  • Vidéos HD Hébergées
  • Tableau de Bord Apprenant
  • Certificats Digitaux
  • Support Chat 24/7

Stack Technologique

Frontend

React, Redux, Tailwind CSS

Backend

Node.js, Express, MongoDB

Média

Cloudinary, Video Streaming

Paiements

Stripe, SafeMoney
