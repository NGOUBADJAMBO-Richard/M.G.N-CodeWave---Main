À Propos du Projet

Waz'UP est une super-app mobile développée en Flutter, combinant e-commerce et services de livraison. L'application offre une expérience utilisateur fluide avec 50K+ téléchargements et une note de 4.8/5.

Défis & Solutions

Défi : Développer une Super-app Performante

Créer une application cross-platform avec e-commerce, paiements en temps réel et géolocalisation.

Solution : Flutter + Backend Robuste

Flutter pour le frontend performant, Node.js pour le backend, MongoDB pour les données, intégration GPS réelle et multiple passerelles de paiement.

Fonctionnalités Principales

  • Catalog E-commerce Dynamique
  • Géolocalisation Temps Réel
  • Paiements Multiples (Stripe, SafeMoney)
  • Système de Notation & Avis
  • Chat Support Intégré
  • Notifications Push Temps Réel

Stack Technologique

Mobile

Flutter, Dart, GetX State Management

Backend

Node.js, Express, MongoDB

Paiements

Stripe API, SafeMoney Gateway

Géolocalisation

Google Maps API, Real-time Tracking
